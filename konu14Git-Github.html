Git ve GitHub Kullanimi

Git :versiyon kontrol sistemidir.

GitHub : git sistemini kullanarak projelerimizi yonetip depolayan uygulamadir.

Onemli Git Terimleri

Repository Kisaca repo, dosyalarimizin bulundugu projeler
Master Projemizin kararli surumu, ana govdesi
Branch Git,agac mantigi ile calisir ve proje birden fazla dala ayrilip bu sekilde gelistirilebilir. Sonrasinda bu dallar yine master da birlestirilip proje tamamlanir
Merge Branch'lerdeki degisikliklerin master da birlestirilme islemidir.
Push Bilgisayardaki kodlarimizi github daki repoya (projenin kendi deposuna)gondermemizi saglar
Pull Github daki projenin son halini bilgisayara cekmeyi saglar
gitignore gitin takip etmesini istedigimiz dosyalari (lokalde denedigimiz resimler vb)push isleminde repoya gonderilmesini engellememizi saglar
